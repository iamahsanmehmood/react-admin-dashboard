> A modern, dark-themed React admin dashboard with charts, data tables, sidebar navigation, and responsive layout. Built with Vite and TailwindCSS for lightning-fast development.

## ✨ Features

- 🌙 **Dark theme** with sleek gradients and glassmorphism
- 📊 **Dashboard cards** — Revenue, Orders, Users, Active sessions
- 📈 **Charts** — Line, Bar, and Doughnut charts with Chart.js
- 📋 **Data tables** — Sortable, searchable user/order tables
- 🗂️ **Sidebar navigation** — Collapsible with icons
- 📱 **Fully responsive** — Desktop, tablet, and mobile
- ⚡ **Blazing fast** — Vite + React + TailwindCSS

## 🚀 Quick Start

```bash
git clone https://github.com/iamahsanmehmood/react-admin-dashboard.git
cd react-admin-dashboard
npm install
npm run dev
```

## 🗂️ Project Structure

```
src/
├── App.jsx                # Main layout with sidebar
├── components/
│   ├── Sidebar.jsx        # Collapsible sidebar navigation
│   ├── StatsCard.jsx      # Dashboard metric cards
│   ├── RevenueChart.jsx   # Line/bar chart component
│   └── DataTable.jsx      # Sortable data table
├── pages/
│   ├── Dashboard.jsx      # Main dashboard view
│   └── Orders.jsx         # Orders management
├── data/
│   └── mockData.js        # Sample data for charts/tables
├── main.jsx
└── index.css              # TailwindCSS + custom styles
```
